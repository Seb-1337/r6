🎮Setup▸
Hold R Ctrl + the number key beside each operator to load their config — works as-is, nothing to remap.Xbox home button = R Alt (already bound).Movement comes set to controller stick — in the config bindings, switch Movement to WASD (it’s the last option) if you play keyboard.
USE UPDATED ROTATION — in the R6 in-game settings, set rotation to Updated — on Classic your aim will feel off. And when creating a new config in XIM Manager, select Updated rotation there too. Every sensitivity number on this page assumes Updated.
Navigation Mode — press [ + ] at the same time to enter navigation mode (XIM lights up white). Then press 1–6 to jump to that Gamepack. Press [ + ] again to exit.Use ] to scroll forward and [ to scroll backward through configs while in navigation mode.
Files preview mode — scripts are off here, so Copy, Select, Search and Expand-all are hidden. To copy a config right now: open SAB Configs → Show code → long-press the code (it selects everything) → Copy. To get Search + one-tap buttons: tap Share and open this file in Chrome (opens local files perfectly) — or any browser app that appears.
🔌How to Plug In▸
1 dot — MouseYour mouse goes in the port marked with one dot.
2 dots — Keyboard / Hub / JoystickKeyboard, USB hub, or a movement joystick goes in the two-dot port.
3 dots — Controller ORIGINAL ONLYA real, official console controller goes in the three-dot port. The console checks it to authenticate — fake or third-party pads won’t pass.
After plugging in: paste the gamepack code into XIM Manager. If you play mouse & keyboard, open the config bindings and switch Movement → WASD — it’s the last option in the list. The config ships on controller-stick movement, so skip this only if you actually move with a stick. When creating the config, select Updated rotation — and set R6’s in-game rotation to Updated as well.
📘SAB — Simple Guide▸

What is SAB? Siege watches how everyone aims. SAB makes your mouse aim look like a controller stick to the game, so you don’t get flagged.

Do I need to change anything? No. These configs already come with safe SAB values. Paste the code and play.

Want snappier aim (lower SAB)? Only ever test on a smurf account, never your main:

  1. Lower Aim SAB by 5 after every 15 clean quickplay games (no detection).
  2. Got detected? Raise it by 10 and start counting again.
  3. Detected in under 10 games → raise by 15. Under 20 → raise by 10. Under 50 → raise by 3.
  4. 100 clean games in a row? That’s your floor — add 2–3 back on to stay safe.

Why so careful? Get flagged as mouse & keyboard 3 times and Siege bans the account for over a year. Never experiment on your main.

Movement & recoil: Siege doesn’t scan movement right now, so keyboard movement is fine. Recoil detection is coming later this year — keep your config updated.
